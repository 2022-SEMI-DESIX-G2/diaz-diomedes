Que es JavaScript?

Javascript es un lenguaje de programación que nos permite dotar a nuestros sitios web de interactividad. A diferencia de HTML, que es un lenguaje de etiquetas que se encarga de organizar la información y mostrarla, Javascript permite que el usuario interactúe con esa información. En resumen, con Javascript podremos convertir una web en una auténtica aplicación con infinidad de capacidades.
